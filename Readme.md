##Simple VPK Icon Shell Extension


This is a simple shell icon extension for psvita *.vpk.

###[Features]
Display psvita vpk icon in explorer

Display vpk type (mai or vitamin)

Show warning overlay icon for unsafe eboot.bin

(unsafe eboot is not a virus,but it could access dangerous system api, use on your own risk. Please don't install unsafe vpk from Unknown sources.)

Simple VPK Icon Shell Extension

这是一个PSV *.vpk文件的windows图标小扩展。

###[功能]
在资源管理器中显示vpk图标

显示VPK类型 (mai 或者 vitamin)

对非安全eboot显示警告图标

（非安全eboot并不是病毒的意思，但是这些应用会调用危险的系统api，请小心使用。千万不要安装奇怪来源的vpk，如果你不知道它是不是病毒。）
